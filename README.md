# Installation

**ATTENTION**: Before proceeding backup your `~/.bashrc`, `~/.vimrc`, `~/.vim`
and any other dotfiles you might want to save.

    git clone git@github.com:calind/dotfiles.git ~/.dotfiles
    cd ~/.dotfiles
    ./install.sh
